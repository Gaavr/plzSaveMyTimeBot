# plzSaveMyTimeBot

Telegram bot that logs messages into Google Sheets.  
Used for tracking time spent on different activities.

## How it works

1. Send a message to the bot in Telegram  
2. The bot parses the message  
3. Data is appended to a Google Sheets table

## Tech

- Java
- Telegram Bot API
- Google Sheets API
